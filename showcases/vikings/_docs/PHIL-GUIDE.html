ℹ️ What you'll see
The spreadsheet has three tabs at the bottom: Fixtures, Standings, and Teams. You only ever need to touch the Fixtures tab. The other two update automatically.

Day-to-Day — How to Update

📅 Adding an upcoming game

Go to the Fixtures tab and add a new row. Fill in Date, Team, Opponent, Venue, and Time. Leave Score and Result blank — fill those in after the game.

🏆 Recording a result

Find the game row. Type the score (e.g. 52-48) and pick the result from the dropdown. Standings update instantly and the website refreshes within a minute.

⚡ Result dropdown
Click the cell in the Result column and a dropdown appears automatically. Pick W (Win), L (Loss), or D (Draw). No typing needed!
⚡ Score format
Always enter the Vikings score first, then a dash, then the opponent's score. For example: 52-48 means Vikings scored 52, opponent scored 48.
ℹ️ Standings are fully automatic
You never need to touch the Standings tab. Every time you add a result in the Fixtures tab, the standings recalculate automatically. The website picks up the changes within a minute or two.
⚠️ Don't change the Standings or Teams tabs
These are managed automatically. If you edit them manually the auto-calculation may overwrite your changes. Everything goes through the Fixtures tab.
